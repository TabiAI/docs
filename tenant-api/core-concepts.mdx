## Locations

A location represents an airport used as an origin or destination.

Locations are identified using their **IATA airport code**.

Example:

| Airport | IATA code |
|---|---|
| John F. Kennedy International Airport | JFK |
| Los Angeles International Airport | LAX |
| Heathrow Airport | LHR |

Use the Locations endpoint to search and validate airports.

## Passengers

Passengers represent the travelers included in a flight search.

Each passenger must specify a supported type.

Supported types:

| Type | Description |
|---|---|
| adult | Passenger aged 12 years or older |
| child | Passenger aged 2–11 years |
| infant | Passenger under 2 years old |

Example:

```json
{
  "passengers": [
    { "type": "adult" },
    { "type": "child" }
  ]
}
```
