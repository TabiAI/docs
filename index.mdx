    ### Welcome to the Travelbase

    Use the Travelbase Tenant API to integrate travel offers, create and manage orders, issue tickets, and receive real-time webhook events.

    The API is designed for secure, reliable server-to-server integrations and supports both sandbox and live environments.
</Note>
